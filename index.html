Game Sorter
Game Sorter
HTML, CSS, Javascript
This program was designed to take the input of a list of the users favorite games. The user is then directed through a series of 1 v 1 comparisons in order to rank the games by preference.
• User inputs a list of favorite games
• Program randomizes input list then places first item into new list
• Next item in original list is compared to midpoint of final list, constricting final location of compared item
• Item is compare to midpoint of new list segment
• Process is repeated until item has only 1 possible location
• Overall process is repeated for entire list
• Results are displayed in numbered list
