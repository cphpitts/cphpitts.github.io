Python | Django
This application allows a user to track their collection of board games. Users can enter data manually as well add games from a list of the hottest games from BoardGameGeek.com. This feature uses BoardGameGeek's API which returns an XML file which is then parsed and added to the database. There is also a news page that uses BeautifulSoup to scrape and parse the newest articles from DiceTowerNews.com.
